# Tasca 2011 DPLL — teaching site

Vite + React + TypeScript + Tailwind. Per-figure pages that explain the
behavioral DPLL simulator under `../sim/`.

Every plot displayed here is generated by our Python simulator
(`scripts/run_*.py`) — none of the original paper&apos;s figures or
text are reproduced.

## Run

From this `web/` directory:

```
npm install
npm run dev
```

Vite serves at http://localhost:5173/.

### Refreshing simulator plots

The PNGs in `public/figures/` are copies of `../results/figures/*.png`.
The copy runs automatically before `npm run dev` and `npm run build`
via the `predev` / `prebuild` hooks. To refresh manually after rerunning
the Python scripts:

```
npm run copy:figures
```

If a plot is missing, the page renders a "Run the matching script and
then copy:figures" notice in place of the image.

## Pages

| Slug                    | Topic                                          |
|-------------------------|------------------------------------------------|
| `/fig-architecture`     | Top-level block diagram and what each block does. |
| `/fig-dtc-cancellation` | Why a 1-bit TDC can work, and what the DTC cancels. |
| `/fig-bbpd-linearize`   | Derivation of `K_bb = sqrt(2/pi)/sigma`.       |
| `/fig-lock-transient`   | Type-II PI loop pulling a 2 MHz frequency offset. |
| `/fig-phase-noise`      | Closed-loop L(f) and integrated RMS jitter.    |
| `/fig-frac-spur`        | Fractional-spur emergence under DTC impairment.|
| `/fig-lms`              | LMS DTC-gain calibration.                      |

## Disclaimer

Every page is labeled "behavioral approximation". The simulator&apos;s
defaults (DCO PN template, loop gains, DTC LSB) are estimated; only
abstract-level numbers and the architectural decomposition are
cross-checked against external sources. See `../docs/assumptions.md`
for the audit trail.
